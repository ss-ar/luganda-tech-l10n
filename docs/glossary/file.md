## Meaning

A digital document or data container stored on a disk.

## Notes

Standard Luganda phonetic adaptation.

Use **Fayiro** consistently.
Do not use alternative spellings such as:

* File
* Fayil
* Fayiroo

Must refer to digital files in technical contexts.

## Examples

* English: "Open file."

* Luganda: "Ggulawo fayiro."

* English: "Save this file."

* Luganda: "Tereka fayiro eno."
