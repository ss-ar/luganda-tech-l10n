## ðŸ”¤ What is l10n?

**l10n** stands for *localization*.

Localization means adapting software to a specific language and cultural context. In this case, we focus on **Luganda (ISO 639-1: lg, ISO 639-2/3: lug)**.

Localization includes:

* Translating user interfaces
* Standardizing technical terminology
* Defining style and grammar guidelines
* Ensuring consistency across platforms
